1. ಆಶೀರ್ವಚನ : ಪರಮಪೂಜ್ಯ ಜಗದ್ಗುರು ಶ್ರೀ ಶ್ರೀ ಭಾರತೀತೀರ್ಥ ಮಹಾ ಸ್ವಾಮಿಗಳವರು, ಶ್ರೀ ಶಾರದಾಪೀಠ, ಶೃಂಗೇರಿ
    2. ಆಶೀರ್ವಚನ : ಪೂಜ್ಯ ಶ್ರೀ ಶ್ರೀ ಸಚ್ಚಿದಾನಂದ ಸರಸ್ವತೀಸ್ವಾಮಿಗಳವರು “ಶ್ರೀಮಠ” ಹರಿಹರಪುರ.
  2. ಪ್ರಸ್ತಾವನೆ
  3. ಮಾಂಡೂಕ್ಯೋಪನಿಷತ್ತು
  4. ಈ ಮಂತ್ರಗಳ ಸರಳವಾದ ಅರ್ಥ
    1. ಮಾಂಡೂಕ್ಯ ಪ್ರವಚನ : ಆಗಮ ಪ್ರಕರಣ-೧
    2. ಮಾಂಡೂಕ್ಯ ಪ್ರವಚನ : ವೈತಥ್ಯ ಪ್ರಕರಣ-೨
    3. ಮಾಂಡೂಕ್ಯ ಪ್ರವಚನ : ಅದ್ವೈತ ಪ್ರಕರಣ-೩
    4. ಮಾಂಡೂಕ್ಯ ಪ್ರವಚನ : ಅಲಾತ ಶಾಂತಿ ಪ್ರಕರಣ-೪
    1. ಮಾಂಡೂಕ್ಯ ಪ್ರವಚನ : ಅನುಬಂಧ-೧ - ಈ ಗ್ರಂಥದಲ್ಲಿರುವ ನಾಲ್ಕು ಪ್ರಕರಣಗಳ ತಾತ್ಪರ್ಯ ಸಂಗ್ರಹ
    2. ಮಾಂಡೂಕ್ಯ ಪ್ರವಚನ : ಅನುಬಂಧ-೨ - ಬೌದ್ಧಮತ ಮತ್ತು ಅದ್ವೈತ ಸಿದ್ಧಾಂತ

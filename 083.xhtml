1. ಮುನ್ನುಡಿ
  2. ಪ್ರಸ್ತಾವನೆ
  3. ಸಮರ್ಪಣೆ
  4. ಧ್ಯಾನಶ್ಲೋಕಗಳು
  5. ಅಧ್ಯಾಯ ೧ಅರ್ಜುನವಿಷಾದಯೋಗ : ಪಾಂಡವರ ಮತ್ತು ಕೌರವರ ಸೈನ್ಯಗಳ ವಿವರ - ಉಭಯ ಸೈನ್ಯಗಳನ್ನು ನೋಡಿ ಅರ್ಜುನನ ವಿಷಾದ - ಸ್ವಜನರೆಂಬ ಮೋಹ - ಯುದ್ಧದಿಂದ ಕುಲಕ್ಷಯ - ಮುಂತಾದವುಗಳಾಗುತ್ತವೆಯೆಂಬ ಭೀತಿ - ಯುದ್ಧ ಮಾಡಲಾರೆನೆಂದು ಅರ್ಜುನನ ಹೇಳಿಕೆ.
  6. ಅಧ್ಯಾಯ ೨ಸಾಂಖ್ಯಯೋಗ : ಅರ್ಜುನನ ಚಿಂತೆ - ಶ್ರೀಕೃಷ್ಣನು ಅರ್ಜುನನ ಮನೋದೌರ್ಬಲ್ಯವನ್ನು ಹಳಿದು ಬುದ್ಧಿ ಹೇಳುವುದು - ಆತ್ಮಕ್ಕೆ ಸಾವಿಲ್ಲ - ದೇಹವು ಎಂದಿದ್ದರೂ ವಿನಾಶಿ - ಸ್ವಧರ್ಮದ ಪಾಲನೆ ಯುಕ್ತವಾದದ್ದು - ಸಮತ್ವಬುದ್ಧಿಯಿಂದ ಯುದ್ಧ ಮಾಡಿದರೆ ಪಾಪವಿಲ್ಲ - ಸಮತ್ವಬುದ್ಧಿಯನ್ನು ಪಡೆದವರ ಲಕ್ಷಣ.
  7. ಅಧ್ಯಾಯ ೩ಕರ್ಮಯೋಗ : ಸಾಧಕನು ನಿತ್ಯಕರ್ಮಗಳನ್ನು ಮಾಡಲೇಬೇಕು - ಜ್ಞಾನಿಗೆ ಕರ್ಮದ ನಿರ್ಬಂಧವಿಲ್ಲ - ಲೋಕಸಂಗ್ರಹಕ್ಕಾದರೂ ಕರ್ಮಾನುಷ್ಠಾನವು ಆವಶ್ಯಕ - ಕರ್ಮವು ಈಶ್ವರಪ್ರೀತ್ಯರ್ಥವಾಗಿರಬೇಕು.
  8. ಅಧ್ಯಾಯ ೪ಜ್ಞಾನಯೋಗ : ಕರ್ಮಯೋಗ ಸಹಿತವಾದ ಜ್ಞಾನಯೋಗದ ಪರಂಪರಾಕ್ರಮ - ಆತ್ಮಜ್ಞಾನವಾದವನಿಗೆ ಕರ್ಮದ ಬಂಧನವಿಲ್ಲ - ಕರ್ಮವನ್ನು ಅಕರ್ಮವೆಂದೂ, ಅಕರ್ಮವನ್ನು ಕರ್ಮವೆಂದೂ ತಿಳಿಯುವ ಬಗೆ - ಯಜ್ಞಭೇದಗಳು - ಜ್ಞಾನಯಜ್ಞ
  9. ಅಧ್ಯಾಯ ೫ಸಂನ್ಯಾಸಯೋಗ : ಜ್ಞಾನರಹಿತ ಸಂನ್ಯಾಸಕ್ಕಿಂತ ಕರ್ಮಾನುಷ್ಠಾನವು ಉತ್ತಮ - ನಾನು ಕರ್ತನೆಂದುಕೊಳ್ಳುವುದು ಅಜ್ಞಾನ-ಜ್ಞಾನಿಗಳ ಸ್ಥಿತಿ - ಜ್ಞಾನಪ್ರಾಪ್ತಿಗೆ ಧ್ಯಾನ ಯೋಗವು ಸಾಧನ.
  10. ಅಧ್ಯಾಯ ೬ಧ್ಯಾನಯೋಗ : ಯೋಗಾರೂಢನಾಗುವವರೆಗೆ ಕರ್ಮವು ಸಾಧನ - ಯೋಗಾರೂಢನಿಗೆ ಕರ್ಮತ್ಯಾಗ - ಯೋಗಾರೂಢನ ಲಕ್ಷಣ - ಯೋಗವನ್ನು ಪಡೆಯುವ ಉಪಾಯ - ಯೋಗಭ್ರಷ್ಟನ ಗತಿ - ಯೋಗಮಾರ್ಗದ ಶ್ರೇಷ್ಠತೆ.
  11. ಅಧ್ಯಾಯ ೭ಜ್ಞಾನವಿಜ್ಞಾನಯೋಗ : ಪರಮಾತ್ಮನೇ ಜಗತ್ಕಾರಣ - ಈಶ್ವರಭಕ್ತಿ - ಅನ್ಯದೇವತೆಗಳಲ್ಲಿ ಭಕ್ತಿ - ಪರಮಾತ್ಮನ ಸ್ವರೂಪವನ್ನರಿಯುವ ಬಗೆ - ಜನರು ಪರಮಾತ್ಮನನ್ನು ತಿಳಿಯಲಾಗದಿರುವುದಕ್ಕೆ ಕಾರಣ.
  12. ಅಧ್ಯಾಯ ೮ಅಕ್ಷರಬ್ರಹ್ಮಯೋಗ : ಬ್ರಹ್ಮವೆಂದರೇನು? ಈ ರೀತಿಯಲ್ಲಿ ಅರ್ಜುನನು ಕೇಳಿದ ಏಳು ಪ್ರಶ್ನೆಗಳಿಗೆ ಉತ್ತರ - ಓಂಕಾರೋಪಾಸನೆ - ಶುಕ್ಲ ಕೃಷ್ಣ ಎಂಬ ಎರಡು ಮಾರ್ಗಗಳು ಯೋಗಿಯು ಪಡೆಯುವ ಗತಿ.
  13. ಅಧ್ಯಾಯ ೯ರಾಜವಿದ್ಯಾರಾಜಗುಹ್ಯಯೋಗ : ಆತ್ಮನ ಸ್ವರೂಪ - ಜಗತ್ತಿನ ಪರಿಸ್ಥಿತಿ - ಜ್ಞಾನಪೂರ್ವಕವಾದ ಉಪಾಸನೆ - ಅನ್ಯದೇಶಗಳ ಉಪಾಸನೆ - ಭಕ್ತನ ಕರ್ತವ್ಯ - ಭಕ್ತಿಯ ಮಹಿಮೆ.
  14. ಅಧ್ಯಾಯ ೧೦ವಿಭೂತಿಯೋಗ : ಪರಮಾತ್ಮನು ಸರ್ವೇಶ್ವರ - ಅರ್ಜುನನ ಪ್ರಾರ್ಥನೆಯಂತೆ ಶ್ರೀಕೃಷ್ಣನು ತನ್ನ ವಿಭೂತಿವಿಶೇಷಗಳನ್ನು ತಿಳಿಸುವುದು.
  15. ಅಧ್ಯಾಯ ೧೧ವಿಶ್ವರೂಪದರ್ಶನಯೋಗ : ಅರ್ಜುನನ ಪ್ರಾರ್ಥನೆಯಂತೆ ಶ್ರೀಕೃಷ್ಣನು ತನ್ನ ವಿಶ್ವರೂಪವನ್ನು ತೋರಿಸುವುದು - ವಿಶ್ವರೂಪದ ಸಾಮಾನ್ಯ ವರ್ಣನೆ - ವಿಶ್ವರೂಪದ ವಿಶೇಷ ವರ್ಣನೆ - ಅರ್ಜುನನು ಮಾಡಿದ ಸ್ತೋತ್ರ - ಅರ್ಜುನನ ಕ್ಷಮಾಯಾಚನೆ.
  16. ಅಧ್ಯಾಯ ೧೨ಭಕ್ತಿಯೋಗ : ಸಗುಣೋಪಾಸನೆ ಮತ್ತು ನಿರ್ಗುಣೋಪಾಸನೆಗಳಿಗಿರುವ ಭೇದ - ಭಗವಂತನಿಗೆ ಪ್ರಿಯವಾದ ಭಕ್ತನ ವರ್ಣನೆ.
  17. ಅಧ್ಯಾಯ ೧೩ಕ್ಷೇತ್ರಕ್ಷೇತ್ರಜ್ಞಯೋಗ : ಕ್ಷೇತ್ರದ ಸ್ವರೂಪ - ಕ್ಷೇತ್ರಜ್ಞನನ್ನು ತಿಳಿಯುವ ಮಾರ್ಗ - ಕ್ಷೇತ್ರಜ್ಞನ ಸ್ವರೂಪ - ಪ್ರಕೃತಿ ಪುರುಷರ ಸ್ವರೂಪ - ಆತ್ಮನು ಕರ್ತನಲ್ಲ - ಆತ್ಮಜ್ಞಾನ.
  18. ಅಧ್ಯಾಯ ೧೪ಗುಣತ್ರಯವಿಭಾಗಯೋಗ : ಸತ್ತ್ವರಜಸ್ತಮೋಗುಣಗಳ ಸ್ವಭಾವ - ಗುಣಾತೀತನ ಸ್ವರೂಪ - ಗುಣಾತೀತನ ಆಚಾರ.
  19. ಅಧ್ಯಾಯ ೧೫ಪುರುಷೋತ್ತಮಯೋಗ : ಸಂಸಾರವೃಕ್ಷ - ಜೀವನು ಪರಮಾತ್ಮನ ಅಂಶಗಳ - ಶ್ರೀಕೃಷ್ಣನು ತನ್ನ ಶಕ್ತಿ ವಿಶೇಷವನ್ನು ತಿಳಿಸುವುದು - ಕ್ಷರ, ಅಕ್ಷರಗಳ ವಿಭಾಗ - ಶ್ರೀಕೃಷ್ಣನು ಪುರುಷೋತ್ತಮ.
  20. ಅಧ್ಯಾಯ ೧೬ದೈವಾಸುರಸಂಪದ್ವಿಭಾಗಯೋಗ : ದೈವೀಸಂಪತ್ತು - ಆಸುರೀಸಂಪತ್ತು - ಅವುಗಳ ಕಾರ್ಯ-ಅಕಾರ್ಯಗಳ ನಿಶ್ಚಯಕ್ಕೆ ಶಾಸ್ತ್ರವೇ ಪ್ರಮಾಣ.
  21. ಅಧ್ಯಾಯ ೧೭ಶ್ರದ್ಧಾತ್ರಯವಿಭಾಗಯೋಗ : ಶ್ರದ್ಧೆಯು ಸಾತ್ತ್ವಿಕೀ ರಾಜಸೀ ತಾಮಸೀ ಎಂದು ಮೂರು ವಿಧ - ಸಾತ್ತ್ವಿಕರೂ ರಾಜಸರೂ ತಾಮಸರೂ ಮಾಡುವ ಕೃತ್ಯಗಳು - ಅವರ ಆಹಾರ - ಯಜ್ಞ, ದಾನ, ತಪಸ್ಸುಗಳು ಗುಣತ್ರಯದಿಂದ ವಿಭಿನ್ನರೂಪಗಳನ್ನು ತಳೆಯುತ್ತವೆ - ಓಂ, ತತ್, ಸತ್-ಎಂಬ ಶಬ್ದಗಳ ಮಹಿಮೆ.
  22. ಅಧ್ಯಾಯ ೧೮ಮೋಕ್ಷಸಂನ್ಯಾಸಯೋಗ : ತ್ಯಾಗಶಬ್ದದ ಅರ್ಥ - ತ್ಯಾಗದಲ್ಲಿ ಮೂರು ಬಗೆ - ಕರ್ಮಾಕ್ಕೆ ಕಾರಣಗಳು - ಜ್ಞಾನ, ಕರ್ತೃ, ಬುದ್ಧಿ, ಧೃತಿಗಳ ಮೂರು ಭೇದಗಳು - ಚಾತುರ್ವರ್ಣ್ಯ - ಸ್ವಧರ್ಮವೇ ಶ್ರೇಯಸ್ಕರ - ಜ್ಞಾನನಿಷ್ಠೆಯನ್ನು ಸಂಪಾದಿಸುವ ಬಗೆ - ಭಕ್ತಿಯಿಂದ ಆತ್ಮಜ್ಞಾನ - ಶರಣಾಗತಿ - ನನ್ನ ಭಕ್ತನಾಗಿರುವುದೇ ಸರ್ವೋತ್ತಮ - ಅನಧಿಕಾರಿಗೆ ಗೀತೆಯನ್ನು ಉಪದೇಶಿಸಬಾರದು - ಗೀತೋಪದೇಶದ ಪರಮ ಪ್ರಯೋಜನ - ಯುದ್ಧವನ್ನು ಮಾಡಲು ಅರ್ಜುನನ ಒಪ್ಪಿಗೆ - ಉಪಸಂಹಾರ.
  23. ಪರಿಶಿಷ್ಟ
  24. ಶ್ಲೋಕಾನುಕ್ರಮಣಿಕೆ
